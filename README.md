# Abstract

Transformer architectures dominate spatiotemporal modeling, from image recognition to video generation and weather forecasting. Transformers exhibit striking in-context learning (ICL) capabilities when trained autoregressively. Recent theoretical work explains these capabilities through mechanistic constructions like induction heads, but is largely confined to 1D token sequences, leaving the spatial generalization of ICL poorly understood. We study \emph{spatial in-context learning} in the setting of cellular automata: given a spatiotemporal trajectory of an unknown local rule, presented as a flattened 1D token sequence, the model must infer and apply the rule without any explicit description. We formalize \emph{spatial induction heads}, two-layer attention circuits that solve this task by decoupling spatial neighborhood routing from content-based retrieval. Theoretically, we introduce multi-peaked spatial-aware embeddings to prove that the required representation dimension is independent of the total grid size, and we characterize a fundamental routing-decoding tradeoff: concentrating routing into fewer attention heads shifts the decoding burden onto the Layer 1 MLP, whose hidden width must scale exponentially with cells routed per head. Empirically, under a strict out-of-sample evaluation, we show that SGD with standard learned positional embeddings naturally converges to this predicted circuit, achieving near-perfect generalization to unseen rules across 1D elementary cellular automata, 2D Von Neumann neighborhoods, and expanded state spaces.
